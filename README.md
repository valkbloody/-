Текствоый редактор для процессора
<br>Название: Разработка текстового редактора с графическим интерфейсом
<br>Цель работы: Разработать приложение – текстовый редактор, который в дальнейшем будет дополнен функциями языкового процессора. Приложение должно иметь графический интерфейс пользователя с возможностью редактирования текста, сохранения/загрузки файлов и <br>вывода результатов работы анализатора.
<br><br>
Сведения об авторе: Зовут Плахин Даннил АП-327 
<br><br>
Описание проекта: Реализован текстовый редаткор с базовым функионалом и графическим пользовательским интефейсом. Для удобства пользователя были реализованы дополнительные функции: возможность изменения размеров текста
<br>выбор языка интерфейса приложения, нумерация строк, открытие файла при перетаскивании иконки в окно программы и т.п.
<br><br>
Используемые технологии: Язык пртограммирования: С#. Среда для графического интерфейса: Windows Forms. IDE: Visual Studio 2022.
<br>
Инструкция по сборке и запуску: 20 мб свободного места на твердотельном накопителе.
<br>
Точные шаги для установки зависимостей: Чтобы открыть редактор без IDE, нужно открыть Текстовый редактор для языкового процессора.exe.
