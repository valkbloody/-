Название: Разработка текстового редактора с графическим интерфейсом
Цель работы: Разработать приложение – текстовый редактор, который в дальнейшем будет дополнен функциями языкового процессора. Приложение должно иметь графический интерфейс пользователя с возможностью редактирования текста, сохранения/загрузки файлов и вывода результатов работы анализатора.

Сведения об авторе: Зовут Плахин Даннил АП-327 

Описание проекта: Реализован текстовый редаткор с базовым функионалом и графическим пользовательским интефейсом. Для удобства пользователя были реализованы дополнительные функции: возможность изменения размеров текста
выбор языка интерфейса приложения, нумерация строк, открытие файла при перетаскивании иконки в окно программы и т.п.

Используемые технологии: Язык пртограммирования: С#. Среда для графического интерфейса: Windows Forms. IDE: Visual Studio 2022.

Инструкция по сборке и запуску: 20 мб свободного места на твердотельном накопителе.

Точные шаги для установки зависимостей: Чтобы открыть редактор без IDE, нужно открыть Текстовый редактор для языкового процессора.exe.

Путь к готовому исполняемому файлу. Папка проекта

Описание интерфейса и функций (руководство пользователя): скриншоты работы программы и интерфейса с пояснением ключевых элементов с подробным указанием, как они вызываются (меню, кнопка, горячая клавиша) и что делают.
